# bitRAKE — Publicity Image Pack (Draft)

This pack is built from the **bitRAKE mark** (crossed drill-bit + rake) and a consistent “metal + graphite + subtle neon” theme.

## Contents

### 01_logo/
- Transparent mark PNGs (2048/1024/512/256)
- White/black mono variants
- White outline variant (good for emboss/laser/etch mockups)

### 02_avatars/
- Square avatars (dark + light), 1024/512/256

### 03_banners/
- X/Twitter: 1500×500
- LinkedIn: 1584×396
- Discord/General: 1920×1080
- YouTube: 2560×1440

### 04_wallpapers/
- 4K + 1440p wallpapers (dark + light)
- Optional “text” wallpaper variant included

### 05_stories/
- 1080×1920 vertical story versions (dark + light)

### 06_patterns/
- Background patterns (2048×2048) and small tiles (512×512)

## Notes
- The **wordmark text** in banners/wallpapers uses a system font as a placeholder (you can swap to your brand font later).
- All assets are PNG (RGBA where relevant).
